🔹 Morning Session – Python Operators
🔑 Key Points

Arithmetic Operators:

+ (addition), - (subtraction), * (multiplication), / (division)

% (modulus → remainder), // (floor division), ** (exponent → power)

Relational Operators (comparison):

>, <, >=, <=, ==, !=

Logical Operators:

and, or, not

💻 Example Code
a = 10
b = 3

# Arithmetic
print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333...
print(a % b)   # 1
print(a // b)  # 3
print(a ** b)  # 1000

# Relational
print(a > b)   # True
print(a == b)  # False
print(a != b)  # True

# Logical
x = True
y = False
print(x and y)  # False
print(x or y)   # True
print(not x)    # False
